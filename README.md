# 🍽️ Restaurant Rating Prediction using Machine Learning

## 📌 Project Overview

The objective is to build a Machine Learning model that predicts a restaurant's **aggregate rating** based on various restaurant features such as location, cuisines, price range, online delivery, table booking, and customer votes.

## 🎯 Objective

- Load and preprocess the restaurant dataset.
- Handle missing values.
- Encode categorical features.
- Split the dataset into training and testing sets.
- Train multiple regression models.
- Compare model performance.
- Identify the most important features affecting restaurant ratings.
